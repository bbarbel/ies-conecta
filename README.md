# IES-Conecta

En este repositorio dejo el código y parte de la documentación técnica de mi proyecto IES-Conecta.

Aquí están el plugin principal de reservas, la base de datos, algunos scripts y varios archivos de apoyo para explicar mejor cómo está montado el sistema.

La idea del proyecto es facilitar la reserva de aulas y recursos dentro del centro, controlando el acceso con Directorio Activo, los permisos según grupos, la validación con QR y el registro de acciones en auditoría.

La versión subida a GitHub está adaptada para poder compartirla, así que no lleva contraseñas reales ni datos sensibles.
