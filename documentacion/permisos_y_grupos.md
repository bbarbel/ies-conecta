# Permisos y grupos

El sistema utiliza los grupos del Directorio Activo para controlar qué puede hacer cada usuario.

## Alumnado

El alumnado no puede acceder al sistema de reservas como profesorado.

Se bloquea su acceso a:

- cuadrantes de reserva
- validación por QR
- páginas de reservas
- auditoría

## Permisos por edificio

Cada usuario puede tener acceso a uno o varios edificios según los grupos a los que pertenezca.

También existen excepciones, como jefatura o profesorado itinerante, con permisos más amplios.

## Permisos por recurso

Además del edificio, el sistema comprueba si el usuario puede reservar un recurso concreto.

Esto permite limitar determinadas aulas o espacios a departamentos específicos.

## Resultado final

Gracias a esta lógica no todos los usuarios ven ni reservan lo mismo, y el sistema se adapta a la organización real del centro.
